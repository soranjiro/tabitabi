# Walica連携

割り勘計算サービス「Walica」との連携機能です。

## Walicaとは？

[Walica](https://walica.jp/) は、グループでの支払いを簡単に割り勘計算できるWebサービスです。

## 概要

しおりとWalicaを連携することで、旅行の割り勘計算がスムーズになります。

## 対応テーマ

この機能は以下のテーマで利用可能です。

- **AI Generated**: 完全対応

## 連携方法

### WalicaのIDを登録

1. Walicaでイベント（旅行）を作成
2. WalicaのイベントURLをコピー
3. しおりの設定画面を開く
4. WalicaのURLまたはIDを貼り付け
5. 保存

### URLの形式

```
https://walica.jp/events/[イベントID]
```

## 連携後の機能

### しおりからWalicaへジャンプ

1. しおり内の「割り勘」ボタンをタップ
2. Walicaのイベントページが開く
3. 支払いを登録

### 表示される情報（テーマによる）

- 現在の合計金額
- 参加者ごとの支払額
- 精算が必要な金額

## 活用の流れ

```
1. 旅行前
   └─ Walicaでイベント作成 → しおりと連携

2. 旅行中
   └─ 支払いがあったらWalicaに登録

3. 旅行後
   └─ Walicaで精算金額を確認 → 送金
```

## メリット

- しおりと割り勘を一元管理
- 旅行中の支払い登録がスムーズ
- グループ全員がリアルタイムで確認可能
