# テーマの選び方

> [!NOTE]
> このドキュメントは作成中です。内容が実際の機能と異なる場合があります。

「たびたび」では、旅のスタイルに合わせて様々なテーマを選べます。

## 利用可能なテーマ

### Minimal（ミニマル）
**シンプルで使いやすい**

- **特徴**: 必要最小限の機能のみ
- **こんな人におすすめ**: シンプルが好き、すぐに使いたい
- **機能**:
  - タイトルと日程の表示
  - 基本的な予定管理
  - メモ機能

---

### Standard Autumn（秋の旅）
**情緒あふれる秋の旅行に**

- **特徴**: 紅葉や秋の雰囲気を演出するデザイン
- **こんな人におすすめ**: 京都や温泉旅行、秋の旅
- **機能**:
  - 詳細な予定管理
  - メモ機能
  - Markdown対応のメモ

---

### AI Generated（AI生成）
**AIが作ったユニークなデザイン**

- **特徴**: 実験的なデザイン、紫基調のモダンなUI
- **こんな人におすすめ**: 個性的なデザインが好き、多機能を求める方
- **機能**:
  - タイムライン形式の予定管理
  - メモ機能
  - シークレットモード
  - Walica連携
  - Markdown対応

---

### Shopping（ショッピング）
**買い物リストに特化**

- **特徴**: 買い物や購入予定の管理に最適
- **こんな人におすすめ**: ショッピング旅行、お土産リスト
- **機能**:
  - リスト形式の予定管理
  - シンプルなUI

---

### Pixel Quest（ピクセルクエスト）
**レトロゲーム風の楽しいデザイン**

- **特徴**: ドット絵のゲームのような見た目
- **こんな人におすすめ**: 楽しい雰囲気が好き、子供と一緒に
- **機能**:
  - クエスト形式の予定管理
  - メモ機能
  - 達成感のある表示

---

## テーマの選び方

### 1. 旅のスタイルで選ぶ

| 旅のタイプ | おすすめテーマ |
|----------|------------|
| 観光・散策 | Standard Autumn |
| ショッピング | Shopping |
| シンプルな記録 | Minimal |
| 遊び心満載 | Pixel Quest |
| サプライズ旅行 | Standard Autumn (シークレットモード) |

### 2. 必要な機能で選ぶ

| 機能 | 対応テーマ |
|------|----------|
| Markdown | Standard Autumn, AI Generated |
| シークレットモード | AI Generated |
| Walica連携 | AI Generated |

### 3. デザインの好みで選ぶ

- **シンプル派**: Minimal
- **和風・情緒的**: Standard Autumn
- **カラフル・楽しい**: Pixel Quest
- **実用的**: Shopping
- **ユニーク**: AI Generated

## テーマの切り替え

作成後でもテーマは変更できます:

1. しおりを編集モードで開く
2. テーマ切り替えボタンをクリック
3. 好きなテーマを選択

### 注意点

- テーマを変更すると、一部の機能が使えなくなる場合があります
- テーマ固有のデータ（例: Walica ID）は保持されますが、新テーマで表示されない場合があります
