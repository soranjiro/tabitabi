# メモ

しおりごとに自由にメモを残せる機能です。

## 概要

旅行の準備や思い出など、なんでも自由に書き込めるメモ欄です。

## 使い方

1. しおりを開く
2. メモセクションを見つける
3. 編集モードで内容を入力
4. 自動保存される

## 活用例

- **持ち物リスト** - 忘れ物チェック
- **予約情報** - ホテルや交通機関の予約番号
- **連絡先** - 緊急連絡先や現地の連絡先
- **注意事項** - 服装、天気、持病の薬など
- **思い出** - 旅行後の感想

## 便利な使い方

### テンプレート例

```
【持ち物】
□ パスポート
□ 財布
□ スマホ充電器

【予約情報】
ホテル: ○○ホテル
予約番号: ABC123

【緊急連絡先】
○○さん: 090-xxxx-xxxx
```

## テーマによる違い

- 一部テーマでは [Markdown対応](../themes/markdown.md) あり
- テーマによって表示デザインが異なる
