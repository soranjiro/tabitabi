## たびたびの特徴

| 特徴 | 説明 |
|------|------|
| 🎨 多彩なテーマ | 旅のスタイルに合わせてデザインを選択 |
| 📱 PWA対応 | アプリのようにインストールして使える |
| 🔒 パスワード保護 | 編集権限を管理できる |
| 🔗 URL共有 | リンクを送るだけで共有できる |
| ✨ シンプル操作 | アカウント登録不要ですぐ使える |
