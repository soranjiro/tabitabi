# シンプルカレンダーテーマ

## 概要
シンプルカレンダーテーマは、Google Calendarのような使いやすいインターフェースで旅程の予定を管理できるテーマです。シンプルで直感的なUIUXを提供し、どんなユーザーでも簡単に旅行計画を立てられます。

**テーマID**: `simple-calendar`

## 特徴

### 1. 縦スクロール型のシンプルな設計
- 旅行のしおりをブログを読むような感覚で追える
- 日付ごとに自動的にグループ化された予定を表示
- 各日付の予定数が一目で分かる

### 2. 複数のビューモード
- **リスト表示** (デフォルト): 縦にスクロールして予定を確認
- **週表示**: 特定の週の予定をカレンダー形式で表示
- **月表示**: 月全体の予定をカレンダー形式で表示

### 3. 使いやすいUI
- 最小限の機能に絞った、凝らないシンプルなデザイン
- 直感的な編集・削除ボタン
- パスワード認証対応
- メモ機能対応

### 4. カスタマイズ可能な情報
各予定に以下の情報を保持できます:
- **タイトル** (必須): 予定の名前
- **日付と時刻** (必須): 予定の日時
- **場所** (オプション): 訪問場所
- **メモ** (オプション): 補足情報

## さまざまな用途
このテーマは以下のような旅行計画に最適です:
- **観光地巡り**: 複数の観光地の訪問予定を日時順に管理
- **グルメ旅**: レストラン訪問の予定管理
- **学校の研修旅行**: 各地点の訪問予定の共有
- **出張日程**: ビジネス出張の移動と会議スケジュール

## UI の特徴

### ヘッダー
- 旅程のタイトルが大きく表示
- 旅行期間（開始日～終了日）を表示
- 編集モードの切り替えボタン

### タイムライン表示（デフォルト）
- **日付セクション**: 各日付ごとにセクション分け
  - 日付と予定数を表示するヘッダー
  - その日の全予定をリスト表示
- **予定カード**: 各予定の情報をコンパクトに表示
  - 時刻 (左寄せ、青色で強調)
  - タイトル
  - 場所（📍アイコン付き）
  - メモ（必要な場合）

### 編集機能
- 予定に対して直感的な編集・削除ボタン
- インラインで予定のタイトル、日時、場所、メモを編集可能
- 確認ダイアログで誤削除を防止

## 使用例

### 新しい旅程の作成
1. ホーム画面からテーマを選択
2. 「シンプルカレンダー」を選択
3. タイトルと日程を入力して旅程を作成

### 予定の追加
1. 「+ 予定を追加」ボタンをクリック
2. タイトル、日付、時刻を入力（必須）
3. 場所とメモを必要に応じて追加（オプション）
4. 「追加」ボタンで保存

### 予定の編集
1. 編集する予定にマウスをホバー
2. ✏️ ボタンをクリック
3. 情報を編集して「保存」をクリック

### 予定の削除
1. 削除する予定にマウスをホバー
2. 🗑️ ボタンをクリック
3. 確認ダイアログで「削除」をクリック
