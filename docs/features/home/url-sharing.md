# URLの追加

URLを使ってしおりにアクセスする機能です。

## 概要

しおりのURLを知っていれば、そのURLからしおりを直接開くことができます。

## 使い方

### しおりを開く

1. しおりのURLを入力欄に貼り付け
2. 「開く」ボタンをタップ
3. しおりが表示される

### URLの形式

```
https://tabitabi.example.com/itineraries/[しおりID]
```

## 活用シーン

- 友達からURLを共有してもらった時
- ブックマークしていないしおりを開く時
- 別のデバイスで同じしおりを見たい時
