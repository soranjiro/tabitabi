# はじめに

「たびたび」へようこそ！

## 「たびたび」とは？

「たびたび」は、旅行の計画や思い出を整理するための**しおり管理アプリ**です。

### できること

- 旅程の作成: 日付ごとに予定を整理
- テーマ選択: 旅のスタイルに合わせたデザイン
- 編集保護: パスワードで編集権限を管理
- 簡単共有: URLを送るだけで共有完了
- PWA対応: アプリのようにインストール可能

## アプリの使い方

### 1. アクセス

ウェブブラウザで https://tabitabi.pages.dev にアクセス

### 2. インストール（オプション）

PWAとしてホーム画面に追加すると、アプリのように使えます。

**iOS (Safari)**
1. 共有ボタン（□↑）をタップ
2. 「ホーム画面に追加」を選択

**Android (Chrome)**
1. メニュー（⋮）を開く
2. 「ホーム画面に追加」を選択

### 3. しおりを作成

1. ホーム画面で「新規作成」をクリック
2. タイトルを入力
3. 編集パスワードを設定（オプション）
4. テーマを選択
5. 「作成」をクリック

## よくある質問

### Q. 無料で使えますか？
A. はい、完全無料です。

### Q. アカウント登録は必要ですか？
A. いいえ、アカウント登録なしで使えます。

### Q. データはどこに保存されますか？
A. サーバーに保存されます。編集パスワードを設定すると、他の人は閲覧のみ可能になります。

### Q. オフラインでも使えますか？
A. PWAとしてインストールすると、一部機能はオフラインでも使用できます。
