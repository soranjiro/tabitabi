# 基本的な使い方

しおりの作成から共有まで、基本的な使い方を説明します。

## しおりの作成

### 1. ホーム画面から作成

1. ホーム画面の「新規作成」ボタンをクリック
2. 必要な情報を入力:
   - **タイトル**: 旅行の名前（例: 京都旅行 2024）
   - **編集パスワード**（オプション）: 編集権限を保護
   - **テーマ**: 好みのデザインを選択

3. 「作成」をクリック

## しおりの編集

### 閲覧モードと編集モード

- **閲覧モード**: 誰でも見られる状態
- **編集モード**: 内容を変更できる状態

### 編集モードに切り替え

1. 画面下部の「編集」ボタンをクリック
2. パスワードを設定している場合は入力
3. 編集モードに切り替わります

### 予定の追加

テーマによって異なりますが、基本的な流れは同じです:

1. 日付を選択
2. 時間を入力
3. 場所や内容を入力
4. 「追加」または「保存」をクリック

## しおりの共有

### URLで共有

1. ブラウザのアドレスバーからURLをコピー
2. メールやLINEなどで送信
3. 相手がURLを開くと、しおりが表示されます

### 共有時の注意点

- **閲覧のみ**: パスワードなしでは閲覧のみ可能
- **編集権限**: パスワードを知っている人は編集可能
- **プライバシー**: URLを知っている人は誰でもアクセス可能

## しおりの管理

### 最近の項目

ホーム画面の「最近開いた項目」から、最近アクセスしたしおりに素早くアクセスできます。

### URLの保存

ブラウザのブックマーク機能を使って、しおりのURLを保存しておくと便利です。

## メモ機能

各しおりには自由に使えるメモ欄があります。

1. しおり画面でメモアイコンをクリック
2. テキストを入力
3. 自動的に保存されます

## Tips

### 編集を楽にする
パスワードを一度入力すると、しばらくは再入力不要です（ブラウザに記憶されます）。

### テーマを変更したい
編集モードでテーマ切り替えボタンから変更できます。

### 印刷したい
ブラウザの印刷機能（Ctrl+P / Cmd+P）で印刷できます。テーマによって印刷レイアウトが最適化されています。
