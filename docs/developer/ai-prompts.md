# AI Agent プロンプト集

AI Agentに開発を依頼する際に使える、効果的なプロンプトの例です。
開発したい内容に合わせて、以下のテンプレートを調整して使用してください。

## 新しいテーマを作成する場合

```markdown
Tabitabiの新しいテーマ「[テーマ名]」を作成してください。
デザインのコンセプトは「[コンセプトの説明]」です。

以下の手順で実装をお願いします：

1. `apps/web/src/lib/themes/[テーマID]` フォルダを作成してください。
2. 既存の `standard-autumn` テーマを参考に、必要なコンポーネントと `index.ts` を実装してください。
3. `apps/web/src/lib/themes/index.ts` を編集して、新しいテーマを登録してください。

既存の機能は維持しつつ、デザイン（色、フォント、レイアウト）でコンセプトを表現してください。
```

## 新しい機能を追加する場合

```markdown
Tabitabiに「[機能名]」機能を追加してください。
要件は以下の通りです：

- [要件1]
- [要件2]

以下の手順で実装をお願いします：

1. `apps/api/migrations` に新しいマイグレーションファイルを作成し、必要なテーブルを定義してください。
2. `apps/api/src/services` にロジックを実装（または既存のServiceを拡張）してください。
3. `apps/api/src/index.ts` にAPIエンドポイントを追加してください。
4. `apps/web/src/lib/api.ts` にAPI呼び出し関数を追加してください。
5. UIコンポーネントを作成して、機能を利用できるようにしてください。
```

## ドキュメントを更新する場合

```markdown
`docs` フォルダ内のドキュメントを更新してください。
[変更内容] に合わせて、わかりやすく記述してください。
専門用語は避け、初心者でも理解できるように心がけてください。
```
