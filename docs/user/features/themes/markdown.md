# Markdown対応

メモ欄でMarkdown記法が使える機能です。

## Markdownとは？

Markdown（マークダウン）は、簡単な記号で文章を装飾できる書き方です。プログラマーに人気ですが、初心者でもすぐに覚えられます。

## 対応テーマ

この機能は以下のテーマで利用可能です。

- **Standard Autumn**: メモと予定のノートで対応
- **AI Generated**: メモと予定のノートで対応

## 基本的な書き方

### 見出し

```markdown
# 大見出し
## 中見出し
### 小見出し
```

### 箇条書き

```markdown
- 項目1
- 項目2
- 項目3
```

### 番号付きリスト

```markdown
1. 最初にやること
2. 次にやること
3. 最後にやること
```

### チェックボックス

```markdown
- [ ] まだやっていない
- [x] 完了した
```

### 太字・斜体

```markdown
**太字** で強調
*斜体* で補足
```

### リンク

```markdown
[表示テキスト](https://example.com)
```

## 活用例

### 持ち物リスト

```markdown
## 持ち物チェックリスト

### 必需品
- [x] パスポート
- [x] 財布
- [ ] モバイルバッテリー

### 衣類
- [ ] 着替え（3日分）
- [ ] 上着
```

### 旅程メモ

```markdown
## 1日目

### 午前
**東京駅** を出発

### 午後
[〇〇ホテル](https://example.com) にチェックイン
```

## 注意点

- すべての記法に対応しているわけではない
- 画像の埋め込みは非対応
- テーマによってデザインが異なる
