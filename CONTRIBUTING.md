# Contributing

コントリビューションありがとうございます！詳細は [docs/developer/](docs/developer/) を参照してください。

## セットアップ

```bash
pnpm install
```

## 開発

```bash
pnpm dev
```

## テスト

```bash
pnpm test
```

## プレビュー環境

Pull Requestで変更内容を確認したい場合、PRのコメント欄に `/deploy-preview` と投稿してください。
プレビュー環境がデプロイされ、PRコメントにURLが投稿されます。

## コミットメッセージ

Conventional Commits に従ってください：

```
<type>(<scope>): <subject>
```

詳しくは [docs/developer/](docs/developer/) を確認してください。
