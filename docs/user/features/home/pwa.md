# PWA（プログレッシブウェブアプリ）

アプリのようにインストールして使える機能です。

## PWAとは？

PWA（Progressive Web App）は、Webサイトをスマートフォンやパソコンの「アプリ」のようにインストールできる技術です。

## メリット

- **ホーム画面に追加** - アプリのようにアイコンから起動
- **全画面表示** - ブラウザのアドレスバーが非表示
- **オフライン対応** - 一部機能はネット接続なしで利用可能
- **高速起動** - キャッシュにより素早く表示

## インストール方法

### iPhone（Safari）

1. Safariでサイトを開く
2. 画面下の「共有」ボタンをタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### Android（Chrome）

1. Chromeでサイトを開く
2. 「インストール」バナーが表示されたらタップ
3. または、メニュー → 「アプリをインストール」

### パソコン（Chrome）

1. Chromeでサイトを開く
2. アドレスバー右側のインストールアイコンをクリック
3. 「インストール」をクリック

## 注意点

- インストールしてもデータはクラウドに保存される
- アンインストールしてもしおりのデータは消えない
- 最新版への更新は自動で行われる
