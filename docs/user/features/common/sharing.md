# 共有機能

URLでしおりを簡単に共有できる機能です。

## 概要

しおりのURLを共有するだけで、友達や家族と同じしおりを見ることができます。

## 共有方法

### URLをコピー

1. しおりを開く
2. 「共有」ボタンをタップ
3. URLがクリップボードにコピーされる
4. LINEやメールで送信

### QRコード（対応テーマのみ）

1. しおりを開く
2. 「QRコード」ボタンをタップ
3. 表示されたQRコードを相手に読み取ってもらう

## 共有相手ができること

| パスワード設定 | 閲覧 | 編集 |
|---------------|------|------|
| なし | ○ | ○ |
| あり | ○ | △（パスワードが必要） |

## 共有のコツ

### グループ旅行の場合

1. 代表者がしおりを作成
2. URLをグループLINEで共有
3. パスワードを設定して、編集は代表者のみ
4. または、パスワードなしで全員編集可能に

### サプライズ旅行の場合

1. [シークレットモード](../themes/secret-mode.md) を使用
2. URLを共有
3. 当日まで内容を隠す

## セキュリティについて

- URLを知っている人は誰でも閲覧可能
- 機密情報は記載しない
- 必要に応じてパスワードを設定
