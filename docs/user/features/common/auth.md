# 認証

パスワードによる編集権限の管理機能です。

## 概要

しおりにパスワードを設定することで、第三者による編集を防ぐことができます。

## 仕組み

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  閲覧モード  │ ──→ │ パスワード入力 │ ──→ │  編集モード  │
└─────────────┘     └─────────────┘     └─────────────┘
```

1. **JWT（トークン）** を使った認証方式
2. 認証後、ブラウザにトークンを保存
3. 次回以降は自動でログイン

## パスワードの設定

### 新規作成時

1. しおりを作成
2. 設定画面でパスワードを設定
3. パスワードは空でもOK（誰でも編集可能）

### 既存しおり

1. 編集モードに入る
2. 設定画面を開く
3. パスワードを変更

## パスワードなしの場合

- 誰でも編集できる
- 気軽に共同編集したい場合に便利
