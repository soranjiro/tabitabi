# シークレットモード

サプライズ用に予定の内容を隠せる機能です。

## 概要

シークレットモードを使うと、しおりの一部または全部を「???」などで隠して表示できます。サプライズ旅行の企画に最適です。

## 使い方

### シークレットモードを有効にする

1. 編集モードに入る
2. 設定を開く
3. 「シークレットモード」をON

### 予定ごとに設定

1. 隠したい予定をタップ
2. 「シークレットにする」をON
3. 表示名を設定（例: 「お楽しみ」）

## 表示の違い

### 通常モード

```
10:00 東京駅集合
12:00 ディズニーランド
19:00 花火大会
```

### シークレットモード

```
10:00 東京駅集合
12:00 **secret**
19:00 **secret**
```

## 解除方法

- 編集者のみが解除可能
- 日時指定で自動解除も設定可能
- 旅行当日に順次公開する使い方も

## 活用シーン

- **サプライズ旅行** - 行き先を当日まで秘密に
- **プロポーズ旅行** - 特別な場所を隠す
- **誕生日サプライズ** - イベントを秘密に
