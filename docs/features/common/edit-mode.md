# 編集モード

閲覧モードと編集モードを切り替える機能です。

## 概要

しおりには2つのモードがあります：

| モード | できること |
|--------|-----------|
| 閲覧モード | 内容を見るだけ |
| 編集モード | 内容の追加・変更・削除 |

## モードの切り替え

### 閲覧 → 編集

1. 「編集」ボタンをタップ
2. パスワードが設定されている場合は入力
3. 編集モードに切り替わる

### 編集 → 閲覧

1. 「完了」または「閲覧」ボタンをタップ
2. 閲覧モードに切り替わる

## 編集モードでできること

- ✅ 予定の追加・編集・削除
- ✅ 予定の並び替え
- ✅ メモの編集
- ✅ 設定の変更

## 閲覧モードのメリット

- 誤操作による変更を防止
- UIがシンプルになる
- 共有相手に編集させたくない時に便利

## 自動ログイン

一度編集モードに入ると、トークンがブラウザに保存されます。
次回からは自動的に編集モードに入れます。
