# Contributing

コントリビューションありがとうございます！詳細は [docs/developer/](docs/developer/) を参照してください。

## セットアップ

```bash
pnpm install
```

## 開発

```bash
pnpm dev
```

## テスト

```bash
pnpm test
```

## コミットメッセージ

Conventional Commits に従ってください：

```
<type>(<scope>): <subject>
```

詳しくは [docs/developer/](docs/developer/) を確認してください。
